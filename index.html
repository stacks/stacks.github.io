What is it?

There are two goals for this workshop: first of all it is a workshop in the usual sense of the word, in this case centered around the topic of algebraic stacks. You will work on a single topic in a small group together with a mentor for a week.

The second goal is to learn how to contribute to the Stacks project. You will learn how to use Git and GitHub, and you will learn how to make sure your LaTeX is compatible with the system that the Stacks project uses. The topics discussed at the workshop can form the basis for a new chapter of the Stacks project, one for which you will be the co-author.

For those who are interested, we will also discuss how it is possible to contribute to the software that runs the Stacks project.
